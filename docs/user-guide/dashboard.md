The user dashboard facilitates quick access to all assemblies bookmarked by users.
## Overview
An overview of all bookmarked assemblies is provided as [assembly cards](assembly-card.md). All bookmarked assemblies can be filtered by assembly name using the searchbar.

## Advanced filters :material-account-hard-hat-outline:
Advanced filters allow the user to filter by Taxon and availability of certain analyses. This includes

- BUSCO
- Repeatmasker
- taXaminer

Additional filters enable filtering by availability of:

- Mappings
- Annotations
